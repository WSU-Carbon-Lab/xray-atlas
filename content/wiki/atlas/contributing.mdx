# Contributing datasets

Contributions keep X-ray Atlas scientifically useful. The goal is to pair each spectrum with enough context for interpretation, comparison, and citation.

## Contribution expectations

- Provide complete experimental metadata whenever available
- Include molecule and sample descriptors for queryability
- Review upload values before final submission
- Link publication context when possible for attribution
- Use citation-ready records when reusing hosted data

## What makes a dataset reusable

A spectrum is reusable when an outside group can judge its comparability
without contacting the author. In practice that means the edge and detection
mode are declared, geometry is recorded for angle-resolved data, pre-edge and
post-edge ranges are supplied so normalization can be assessed, uncertainties
are included when the acquisition supports them, and a publication or
contributor record anchors provenance. The
[quality metrics](/wiki/atlas/quality-metrics) page shows how these fields
feed the signals displayed on browse cards.
